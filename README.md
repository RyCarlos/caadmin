
ThinkPHP 6.0 + Layui

> 运行环境要求PHP7.1+。

## 主要特性

* 后端框架 thinkPHP6 前端框架Layui
* 界面模仿Layui官网后台模板
* 对一些Layui的一些常用组件进行了模块化封装
## 安装

~~~
git clone git@github.com:RyCarlos/caadmin.git
~~~
